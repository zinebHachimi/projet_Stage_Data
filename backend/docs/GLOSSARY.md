# Glossary

| Term                  | Definition                                                            |
| --------------------- | --------------------------------------------------------------------- |
| **API Key**           | A secret token included in the `x-api-key` header for authentication  |
| **Cache Expiry**      | Duration (seconds) before cached results are invalidated              |
| **Rate Limiting**     | Restricting the number of requests per time window per client         |
| **NestJS**            | Node.js framework for building server-side applications               |
| **Docker Compose**    | Tool for defining and running multi-container Docker applications     |
| **Swagger / OpenAPI** | Standard for describing REST APIs; used for interactive docs          |
| **Source Module**     | Module that fetches job listings from a specific job board            |
| **ATS**               | Applicant Tracking System — software companies use to manage hiring   |
| **RSS**               | Really Simple Syndication — XML-based feed format for content updates |
| **companySlug**       | Company identifier used by ATS sources to target a specific board     |
| **TTL**               | Time-to-live — how long cached data remains valid                     |
| **Guard**             | NestJS mechanism for request authentication/authorization             |
| **Interceptor**       | NestJS mechanism for transforming requests/responses                  |
| **Throttler**         | NestJS rate limiting module (`@nestjs/throttler`)                     |
| **Monorepo**          | Single repository containing multiple packages/applications           |
| **DTO**               | Data Transfer Object — typed class for API input/output validation    |
| **MCP**               | Model Context Protocol — standard for connecting AI systems to tools  |
| **Scalar**            | Modern API documentation UI, alternative to Swagger UI                |
| **Analytics**         | Module providing job data summary statistics and company intelligence |
